Oppsummering kapittel 4 Internett

  • Internett er et nettverk av nettverk. Internett funger ved at ulike system kobles til hverandre, kommuniserer med hverandre og jobber sammen på grunn av avtalte standarder/protokoller for hvordan data blir sendt rundt på internettet.
  • All informasjon som skal sendes, sendes fra en datamaskin til en annen
  • En klient (en datamaskin) spør om informasjon som ligger på en tjener (også datamaskin).
  • HTTP og DNS styrer sending og mottak av web filer. Dette er mulig ved bruk av TCP, IP og ruternettverk som bryter ned og transporterer informasjon i mindre pakker, samt sjekker at alt er kommet frem.
  • Hvert nettsted på internett har en unik IP-adresse, som ofte skrives ved domenenavn (www.eksempel.com).
  • Selve pakkene består av binære sekvenser av enere og nuller som blir fysisk send gjennom elektriske ledninger, fiberoptiske kabler og trådløse nettverk.

Forkortlser og betydning

HTTP HyperText Transfer Protocol Språket til datamaskiner
DNS Domain Name System Assosierer navn domenavn med tilsvarende IP-adresser
TCP Transmission Control Protocol Ansvarlig for sending av informasjon (pakker) mellom datamaskiner
IP Internettprotokollen Et bestemt regelsett om hvordan informasjon blir sendt fra en data til en annen på Internett
