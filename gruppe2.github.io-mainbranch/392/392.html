Hvordan publisere webside på Github

Her er stegene du må følge for å lage en repository.

Steg 1. Lag en bruker på github.com

Følg denne instruksen til å finne ut hvordan du kan lage en bruker på GitHub.


Steg 2. Lag en ny repository

  1. Klikk på den grønne «New Repository»
  2. Gi den ett navn og gjerne en beskrivelse (dette kan endres senere).
  3. Pass på at den er huket av på «public»! Slik at andre kan se og bidra til din repository.
  4. Huk av på «Initialize this repository with a README.»
  5. Trykk på den grønne «Create repository».

Gratulerer! Du har nå laget din egen repository.
Videre kan du trykke på "New file" for å lage nye filer til arbeidet ditt.
